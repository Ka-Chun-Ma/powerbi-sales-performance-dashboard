# Project Summary for Recruiters

**Sales Performance Dashboard (Power BI)** — Portfolio project built on the Microsoft AdventureWorks sample dataset. Delivers an executive-level view (revenue, profit, orders, trends), a regional map, and drill-down pages for products and customers, plus a what-if style price-adjustment analysis. Demonstrates data modeling in Power BI, DAX measures, parameters, and report design with clear layout and interactivity (filters, drill-down, cross-filtering). Developed as a guided (curriculum-based) project; implementation and formatting done independently. Suitable as evidence of Power BI and analytical reporting skills for Data Analyst roles.
